#BabySchool 
=============
## DESCRIPTION
A Student/Course Management System Implemented with django 
### Procedure
### Simply clone this repo
    $ git clone [repo link]
    $ cd project_name
    $ pipenv shell
    $ pipenv install
    $ python manage.py runserver

change the python_version to your system version in your Pipfile

## Technology Used
* Backend 
    * Django/Python
    * Postgresql
* Frontend
    * HTML/CSS
    * Bootstrap
    * Javascript
